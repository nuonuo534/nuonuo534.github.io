1、清除悬空镜像
```go
docker rmi $(docker images -f "dangling=true" -q)
// 等价于
-a Remove all unused images, not just dangling ones
-f 不提示确认
docker image prune
// it will remove both unused and dangling images.
docker system prune -a  
```
